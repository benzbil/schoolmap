🚀 Deploy ไปยัง Google Apps Script

ใช้โค้ดด้านล่างเพื่อ deploy ระบบแผนที่ไปยัง Google Apps Script

📝 ขั้นตอนการ Deploy:

  1. เปิด Google Apps Script
  2. สร้างโปรเจคใหม่
  3. Copy โค้ด HTML ด้านล่างใส่ในไฟล์ Index.html
  4. Copy โค้ด Apps Script ใส่ในไฟล์ Code.gs
  5. Deploy เป็น Web App

📄 HTML Code (Index.html):

<!DOCTYPE html> <html> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>ระบบแผนที่โรงเรียนอัจฉริยะ</title> <!-- CSS จะถูก generate อัตโนมัติ --> <?= include('styles'); ?> </head> <body> <!-- HTML Content จะถูก generate อัตโนมัติ --> <?= include('content'); ?> <!-- JavaScript จะถูก generate อัตโนมัติ --> <?= include('scripts'); ?> </body> </html>

⚙️ Apps Script Code (Code.gs):
