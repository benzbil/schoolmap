1 Set Up Your Google Sheet

First, we need to create a Google Sheet to store all your school location data.

üìä Create the Sheet:

  1. Go to sheets.google.com in your web browser
  2. Click "Blank" to create a new spreadsheet
  3. Name your sheet: "School Map Locations"
  4. In the first row, create these column headers exactly as shown:
name | description | type | lat | lng | x | y

üìù Add Sample Data:

Copy and paste this sample data into your sheet (starting from row 2):

‚ö†Ô∏è Important: Make sure your column headers are spelled exactly as shown above. The code depends on these exact names!

2 Create Google Apps Script

Now we'll create a script that lets your website read and write data to the Google Sheet.

üîß Create the Script:

  1. In your Google Sheet, click Extensions ‚Üí Apps Script
  2. Delete the existing code in the editor
  3. Copy and paste this code:
// Google Apps Script for School Map Website // This script lets your website read and write to Google Sheets function doGet(e) { // This function runs when someone visits your script URL // It reads all the data from your sheet and returns it as JSON try { // Get the active spreadsheet const sheet = SpreadsheetApp.getActiveSheet(); // Get all the data from the sheet const data = sheet.getDataRange().getValues(); // The first row contains headers, so we skip it const headers = data[0]; const rows = data.slice(1); // Convert each row into an object with named properties const locations = rows.map(function(row) { const location = {}; headers.forEach(function(header, index) { location[header] = row[index]; }); return location; }); // Return the data as JSON return ContentService .createTextOutput(JSON.stringify(locations)) .setMimeType(ContentService.MimeType.JSON); } catch (error) { // If something goes wrong, return an error message return ContentService .createTextOutput(JSON.stringify({error: error.toString()})) .setMimeType(ContentService.MimeType.JSON); } } function doPost(e) { // This function runs when someone sends data to your script // It adds new locations to your sheet try { // Get the data that was sent to the script const data = JSON.parse(e.postData.contents); // Get the active spreadsheet const sheet = SpreadsheetApp.getActiveSheet(); // Add the new data as a new row sheet.appendRow([ data.name, data.description, data.type, data.lat, data.lng, data.x, data.y ]); // Return success message return ContentService .createTextOutput(JSON.stringify({success: true})) .setMimeType(ContentService.MimeType.JSON); } catch (error) { // If something goes wrong, return an error message return ContentService .createTextOutput(JSON.stringify({error: error.toString()})) .setMimeType(ContentService.MimeType.JSON); } }

üöÄ Deploy the Script:

  1. Click the Deploy button (top right)
  2. Choose New deployment
  3. Set Type to Web app
  4. Set Execute as: Me
  5. Set Who has access: Anyone
  6. Click Deploy
  7. Copy the Web app URL - you'll need this!
üîí Security Note: This makes your script public so your website can use it. Only put school location data in the sheet - no personal information!

3 Connect Your Website to the Script

Now we need to tell your website where to find the Google Apps Script.

üìù Update the JavaScript:

  1. Open your script.js file
  2. Find this line near the top:
const GOOGLE_SCRIPT_URL = 'YOUR_GOOGLE_APPS_SCRIPT_URL_HERE';
  1. Replace YOUR_GOOGLE_APPS_SCRIPT_URL_HERE with the Web app URL you copied in step 2
  2. Save the file

Your line should look something like this:

const GOOGLE_SCRIPT_URL = 'https://script.google.com/macros/s/AKfycbx.../exec';

üß™ Test Steps:

5 Publish Your Website

üÜò Troubleshooting Guide

If something isn't working, try these solutions:

‚ùå "Loading..." Never Disappears

  • Check your Google Apps Script URL in script.js
  • Make sure your Google Apps Script is deployed as a web app
  • Verify the script has permission to access your sheet

‚ùå Search Doesn't Find Locations

  • Check that location names in your Google Sheet match what you're searching for
  • Make sure there are no extra spaces in your data
  • Try searching for partial names (e.g., "Library" instead of "School Library")

‚ùå Admin Page Can't Add Locations

  • Verify your doPost function is working in Google Apps Script
  • Check that your sheet has the correct column headers
  • Make sure the script has permission to edit the sheet

‚ùå Website Looks Broken

  • Make sure all files (HTML, CSS, JS) are in the same folder
  • Check that style.css is properly linked in your HTML
  • Clear your browser cache and try again
