คู่มือการ Deploy ระบบ

วิธีการนำระบบไปใช้งานจริงบน GitHub Pages

ขั้นตอนการเตรียมไฟล์

ขั้นตอนที่ 1: เปลี่ยนชื่อไฟล์

เมื่อดาวน์โหลดไฟล์แล้ว ให้เปลี่ยนชื่อดังนี้:

school_map_system_v6.html → index.html
school_map_system_admin_v6.html → admin.html

สำคัญ: ต้องเปลี่ยนชื่อไฟล์ให้ถูกต้องเพื่อให้ GitHub Pages ทำงานได้

ขั้นตอนที่ 2: สร้าง Repository บน GitHub

  • เข้าสู่ GitHub และสร้าง Repository ใหม่
  • ตั้งชื่อ Repository เช่น school-map-system
  • เลือก Public Repository
  • อัปโหลดไฟล์ index.html และ admin.html

ขั้นตอนที่ 3: เปิดใช้งาน GitHub Pages

  1. ไปที่ Settings ของ Repository
  2. เลื่อนลงไปหา "Pages" ในเมนูซ้าย
  3. ในส่วน Source เลือก "Deploy from a branch"
  4. เลือก Branch: main และ Folder: / (root)
  5. คลิก Save

ขั้นตอนที่ 4: แก้ไข URL ในโค้ด

หลังจาก Deploy แล้ว ต้องแก้ไข URL ในไฟล์ทั้งสองเพื่อให้เชื่อมต่อกันได้:

ในไฟล์ index.html หาบรรทัด:

function openAdminInterface() {
    window.open('school_map_system_admin_v6.html', '_blank');
}

แก้เป็น:

function openAdminInterface() {
    window.open('https://[USERNAME].github.io/[REPO]/admin.html', '_blank');
}

ในไฟล์ admin.html หาบรรทัด:

function openUserInterface() {
    window.open('school_map_system_v6.html', '_blank');
}

แก้เป็น:

function openUserInterface() {
    window.open('https://[USERNAME].github.io/[REPO]/index.html', '_blank');
}

แทนที่ [USERNAME] ด้วยชื่อผู้ใช้ GitHub และ [REPO] ด้วยชื่อ Repository

ขั้นตอนที่ 5: ทดสอบระบบ

  • เข้าใช้งานผ่าน URL: https://[USERNAME].github.io/[REPO]/
  • ทดสอบการค้นหาห้องเรียน
  • ทดสอบการตั้งจุดเริ่มต้นและเส้นทาง
  • ทดสอบการเข้าสู่ Admin Panel (admin/1234)
  • ทดสอบการเพิ่ม/แก้ไข/ลบข้อมูล

ข้อกำหนดและข้อแนะนำ

📋 ข้อกำหนดการใช้งาน

  • เบราว์เซอร์ที่รองรับ: Chrome, Firefox, Safari, Edge (เวอร์ชันล่าสุด)
  • ใช้งานได้ทั้งบนคอมพิวเตอร์และมือถือ
  • ไม่ต้องติดตั้ง Server หรือ Database
  • ข้อมูลจัดเก็บใน localStorage ของเบราว์เซอร์

💡 ข้อแนะนำเพิ่มเติม

  • สำรองข้อมูลเป็นประจำผ่านฟีเจอร์ Export ในระบบ
  • ใช้ Google Sheets สำหรับ Live Mode เพื่อการทำงานร่วมกัน
  • ปรับแต่งข้อมูลห้องเรียนให้ตรงกับโรงเรียนจริง
  • ทดสอบระบบอย่างสม่ำเสมอหลังการปรับปรุง

4. ดู logs ใน Console เพื่อหาสาเหตุปัญหา

คู่มือการตั้งค่า Google Sheets แบบสมบูรณ์

ขั้นตอนการ Setup (5 ขั้นตอน)

ขั้นตอนที่ 1: สร้าง Google Sheets

1. เข้า Google Sheets และสร้างไฟล์ใหม่

2. สร้าง 3 แผ่นงาน: Buildings, Rooms และ MapImages

📋 โครงสร้าง Sheet "Buildings":

code | name | floors | year | description | rooms | status | created | updated

📋 โครงสร้าง Sheet "Rooms":

code | name | building | floor | type | description | status | x | y | images | created | updated

📋 โครงสร้าง Sheet "MapImages" (ใหม่):

id | floorplan | realphoto | notes | updated
ขั้นตอนที่ 2: เปิด Google Apps Script

1. ในไฟล์ Google Sheets ไปที่ ส่วนขยาย → Apps Script

2. ลบโค้ดเก่าแล้วใส่โค้ดด้านล่าง:

function doGet(e) { return handleRequest(e); } function doPost(e) { return handleRequest(e); } function handleRequest(e) { const action = e.parameter.action; const sheetName = e.parameter.sheet || 'Buildings'; try { switch(action) { case 'getData': return getData(sheetName); case 'addData': return addData(sheetName, e.parameter); case 'updateData': return updateData(sheetName, e.parameter); case 'deleteData': return deleteData(sheetName, e.parameter.id); default: return ContentService.createTextOutput(JSON.stringify({ success: false, error: 'Invalid action' })).setMimeType(ContentService.MimeType.JSON); } } catch (error) { return ContentService.createTextOutput(JSON.stringify({ success: false, error: error.toString() })).setMimeType(ContentService.MimeType.JSON); } }
ขั้นตอนที่ 3: Deploy Web App

1. คลิก Deploy → New deployment

2. เลือก Type: Web app

3. Execute as: Me

4. Who has access: Anyone

5. คลิก Deploy

ระบบใช้ URL เริ่มต้นแล้ว ไม่ต้องคัดลอก URL ใหม่

ขั้นตอนที่ 4: การตั้งค่าเสร็จสิ้น

ระบบได้ตั้งค่าค่าเริ่มต้นไว้แล้ว:

  • URL: ตั้งค่าแล้ว (ไม่ต้องแก้ไข)
  • Buildings Sheet: "Buildings"
  • Rooms Sheet: "Rooms"
  • MapImages Sheet: "MapImages"
ขั้นตอนที่ 5: ทดสอบการเชื่อมต่อ

1. คลิก "ทดสอบการเชื่อมต่อ" ด้านบน

2. หากสำเร็จ ข้อมูลจะซิงค์แบบเรียลไทม์

เมื่อเชื่อมต่อสำเร็จ ข้อมูลจะอัพเดทใน Google Sheets ทันที
